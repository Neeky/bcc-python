## bcc-python

bcc-python 在 bcc 的基础之上做了一些易用性的提升，用起来简单是我们的目标。

---

## 安装

1、安装 bcc 依赖库
```bash
yum -y install bcc-tools
```
2、安装 bcc-python 
```
pip3 install bcc-python
```
